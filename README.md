# Encrypting-data-at-rest-with-Python
Python program that creates a tool that will provide the ability to encrypt and decrypt data at rest.
• It will be able to encrypt a single file, single folder, multiple files and multiple folders.
• It will support both true cryptographic keys, and password generated keys, and give the user the option to choose 
    • If using a true cryptographic keys, it should give the user the option to have the
  program generate it, or have them provide it in hexadecimal
• Supports AES 128, 192 and 256
• It will be able to decrypt anything that it encrypts, assuming the proper
key/password is provided
