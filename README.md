# Encrypting-data-at-rest-with-Python

Python program that creates a tool that will provide the ability to encrypt and decrypt data at rest.
• Encrypt.py will be able to encrypt a single file, single folder, multiple files and multiple folders.
• Encrypt.py will support both true cryptographic keys, and password generated keys, and give the user the option to choose 
    • If using a true cryptographic keys, Encrypt.py should give the user the option to have the
  program generate it, or have them provide Encrypt.py in hexadecimal
• Supports AES 128, 192 and 256
• Encrypt.py will be able to decrypt anything that Encrypt.py encrypts, assuming the proper
key/password is provided
